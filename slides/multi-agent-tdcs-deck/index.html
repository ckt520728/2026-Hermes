Lessons Learned

7 個坑，全部轉成流程規則

  • 大任務拆成可平行的小任務，避免 timeout。
  • 給 subagent 絕對檔案路徑，不假設它有父對話記憶。
  • 跨 Agent 先定義單位、資料格式與 success criteria。
  • Agent 5 必須拿到 story arc，否則論文像報告拼貼。

AI Agent 不是取代研究者，而是把研究團隊中可結構化、可驗證、可委派的部分放大。
